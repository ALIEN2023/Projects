Для запуска неоюходим SFML 2.5.1 для Windows.

SFML\include добавить в Проект->Свойства->C/C++->Общие->Дополнительные каталоги включаемых файлов

SFML\lib добавить в Проект->Свойства->Компоновщик->Общие->Дополнительные каталоги библиотек

В Проект->Свойства->Компоновщик->ВВод->Дополнительные зависимости добавить sfml-graphics-d.lib; sfml-window-d.lib; sfml-system-d.lib
